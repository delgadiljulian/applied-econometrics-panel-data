## Projects

### Gender Wage Gap and Economic Development in Latin America
An empirical study analyzing whether gender wage gaps increase or decrease as economies develop, using panel data from Latin America and standard fixed effects estimators.

- Methods: OLS, Fixed Effects, Two-Way Fixed Effects
- Data: GenLAC, World Bank
- Tools: Stata
